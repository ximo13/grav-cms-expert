# Themes

## Theme Discovery

For Grav 1.7 themes, use a theme configuration file matching the theme slug:

Example:

```text
user/themes/grav-expert-theme/grav-expert-theme.yaml
```

Do not rely only on `theme.yaml`.

## Minimum Theme Structure

```text
theme/
├── blueprints.yaml
├── grav-expert-theme.php
├── grav-expert-theme.yaml
├── CHANGELOG.md
├── templates/
└── css/
```
