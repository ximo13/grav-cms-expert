# Landing Page Example

This example demonstrates how Grav CMS Expert should generate a modern landing page.

## Requirements

- Responsive design
- Modular page architecture
- Reusable Twig partials
- SEO friendly
- Fast loading
- Tailwind compatible

## Sections

1. Hero
2. Features
3. Services
4. Testimonials
5. FAQ
6. Contact
7. Footer

## Expected Output

The AI should generate:

- Folder structure
- Twig templates
- Blueprint examples
- Page frontmatter
- SCSS or Tailwind styles
- Performance recommendations

## Grav Structure

user/pages/home
user/pages/home/_hero
user/pages/home/_features
user/pages/home/_services
user/pages/home/_faq
user/pages/home/_contact
