# AGENTS.md

## Project Role

You are working on Grav CMS Expert, an open-source AI assistant for Grav CMS developers.

## Main Goal

Help developers create, debug and optimize Grav CMS projects using:

- Twig
- YAML
- PHP
- Grav themes
- Grav plugins
- Modular pages
- Blueprints
- Flex Objects
- SEO
- Performance optimization
- Tailwind
- Gantry 5

## Repository Rules

- Follow Grav CMS best practices.
- Keep examples production-ready.
- Prefer modular architecture.
- Use reusable Twig partials.
- Always explain where files should be placed.
- Keep YAML valid and properly indented.
- Use clean PHP compatible with Grav.
- Avoid unnecessary plugins.
- Update README when adding major features.
- Keep documentation beginner-friendly.

## Important Folders

- `prompts/` contains reusable AI prompts.
- `snippets/` contains reusable code snippets.
- `examples/` contains complete Grav examples.
- `SKILL.md` contains the main skill instructions.
- `AGENTS.md` contains Codex-compatible project instructions.

## Verification Checklist

Before considering work complete:

- Check Twig syntax.

- Update documentation if needed.
